# 건설 주요 자재 원가 분석 보고서

**작성일:** 2026년 3월 2일  
**분석 기간:** 2025년 3월 ~ 2026년 2월 (최근 12개월)  
**작성:** 건설 원가관리 전문 분석

---

## 1. 요약 (Executive Summary)

건설경기 장기 침체 속에서 레미콘과 철근은 **상반된 가격 흐름**을 보이고 있다.

| 자재 | 12개월 전 | 현재 | 변동률 | 방향 |
|------|----------|------|--------|------|
| 레미콘 (수도권, m³) | 93,700원 → 91,400원 | 89,000원 | **-5.0%** | 하락 |
| 철근 (SD400, ton) | 700,000원 | 760,000원 | **+8.6%** | 상승 |

- **레미콘**: 수요 급감(가동률 17.4%, IMF 이하)으로 가격 하방 압력. 2025년 수도권 협상단가 2.45% 인하.
- **철근**: 철스크랩 가격 상승 + 현대제철 인천공장 대보수(3~4월)로 공급 타이트. 2026.3월 톤당 77만원+추가인상 예고.

## 2. 레미콘 시장 분석

### 2.1 가격 추이

수도권 레미콘 장기 추이 (연간 협상 단가, 원/m³):

| 연도 | 단가 | 전년比 |
|------|------|--------|
| 2020 | 66,300 | - |
| 2021 | 67,700 | +2.1% |
| 2022 | 80,300 | +18.6% |
| 2023 | 88,700 | +10.5% |
| 2024 | 93,700 | +5.6% |
| 2025 | 91,400 | **-2.5%** |

2022~2024년 급등 후, 2025년 처음으로 **하락 전환**. 이후 월별로 추가 하락하여 2026.2월 기준 약 89,000원 수준으로 추정.

### 2.2 수급 현황

- **생산량**: 2022년 1.41억m³ → 2023년 1.36억m³ → 2024년 1.12억m³ (3년간 -21%)
- **가동률**: 2023년 21.4% → 2024년 17.4% (IMF 당시 29.6%보다 낮은 **역대 최저**)
- 시멘트 내수: 2026.1~2월 445만톤 (전년 동기 591만톤 대비 **-24.8%**)
- 시멘트 생산라인 35기 중 10기 가동중단, 재고율 90% 육박

### 2.3 핵심 변동 요인

| 요인 | 구분 | 영향도 |
|------|------|--------|
| 건설 착공물량 급감 | 수요 | ★★★ |
| 건설사 법정관리 급증 | 수요 | ★★★ |
| 시멘트 원가 상승 (환경규제) | 공급 | ★★☆ |
| 정부 배치플랜트 규제 완화 | 외부 | ★★☆ |
| 민·관 가격 안정화 협의체 | 외부 | ★★☆ |

## 3. 철근 시장 분석

### 3.1 가격 추이

현대제철 SD400 유통가 기준 (원/ton):

- 2025년 상반기: 690,000~700,000원 (약보합)
- 2025년 하반기: 710,000~735,000원 (상승 전환)
- 2026년 1월: 745,000원
- 2026년 2월: 760,000원 (고시가 76만원)
- 2026년 3월: **770,000원 + 추가인상 예고** (고시가 77만원, 16일부 1~2만원 추가)

12개월 간 약 **+8.6% 상승**.

### 3.2 수급 현황

- 철스크랩 가격 2025년 하반기부터 지속 상승 → 제강 원가 부담
- 현대제철 인천공장: 소형압연 폐쇄 + 철근 압연라인 대보수(3/5~4/3) → 유통향 공급 사실상 중단
- 한수원 원전 프로젝트향 특수 규격 물량 집중 → 일반 유통 물량 제한
- 철근 재고 급감 → 유통 공급 타이트 장기화

### 3.3 핵심 변동 요인

| 요인 | 구분 | 영향도 |
|------|------|--------|
| 철스크랩 가격 급등 | 공급 | ★★★ |
| 인천공장 대보수/폐쇄 | 공급 | ★★★ |
| 건설경기 침체 (수요 약화) | 수요 | ★★☆ |
| 미국 수출 수요 유인 | 외부 | ★☆☆ |
| 원전 프로젝트 특수 수요 | 수요 | ★★☆ |

## 4. 시나리오 분석

총 공사비 100억 기준 (공동주택), 레미콘 비중 10%, 철근 비중 12% 가정.

### 시나리오 A: 낙관 (Optimistic)

| 항목 | 변동률 | 영향액 |
|------|--------|--------|
| 레미콘 | -3% | -0.3억 |
| 철근 | -5% | -0.6억 |
| **합계** | | **-0.9억 ~ -1.8억** |

**전제:** 하반기 건설경기 회복 시그널, 정부 SOC 투자 확대로 물량 회복, 철스크랩 국제가 안정, 에너지 비용 하락.

### 시나리오 B: 기본 (Base)

| 항목 | 변동률 | 영향액 |
|------|--------|--------|
| 레미콘 | +2% | +0.2억 |
| 철근 | +8% | +1.0억 |
| **합계** | | **+1.2억 ~ +2.2억** |

**전제:** 현 추세 유지. 레미콘은 수요 약세 속 소폭 반등(시멘트 원가 전가), 철근은 원가 상승분 지속 전가.

### 시나리오 C: 비관 (Pessimistic)

| 항목 | 변동률 | 영향액 |
|------|--------|--------|
| 레미콘 | +7% | +0.7억 |
| 철근 | +18% | +2.2억 |
| **합계** | | **+2.9억 ~ +5.6억** |

**전제:** 건설경기 추가 침체 + 자재업체 도산으로 공급 축소, 철스크랩·에너지 비용 급등, 환율 악화.

## 5. 리스크 및 시사점

### 5.1 당면 리스크

1. **레미콘-철근 가격 디커플링**: 레미콘은 수요 부진으로 하락, 철근은 원가 상승으로 상승 → 원가 예측 난이도 증가
2. **공급 인프라 훼손**: 레미콘 공장 폐업, 시멘트 라인 중단, 철근 압연 폐쇄 → 경기 회복 시 공급 병목 우려
3. **건설사 연쇄 도산**: 미수금 회수 불능 → 자재업체 연쇄 부실

### 5.2 원가관리 제언

1. **철근 선구매/장기계약**: 2분기 추가 인상 예상되므로 3월 중 물량 확보 권고
2. **레미콘 단가 재협상**: 하반기 추가 인하 여지 있으므로 분기별 재협상 조건 확보
3. **대체재·공법 검토**: 철근 비중 축소를 위한 PC 공법, 고강도 철근(SD500) 활용 검토
4. **에스컬레이션 조항**: 신규 도급계약 시 원자재 가격 연동 조항 필수 반영
